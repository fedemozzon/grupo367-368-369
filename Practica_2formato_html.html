Listas ordenadas

La lista debe ir desde la letra "a" hasta la "d"

a. Elemento 1 b. Elemento 2 c. Elemento 3 d. Elemento 4

La lista debe ir desde la letra "g" hasta la "j"

g. Elemento 1 h. Elemento 2 i. Elemento 3 j. Elemento 4

La lista debe ir desde la letra "z" hasta la "w" (descendente)

z. Elemento 1 y. Elemento 2 x. Elemento 3 w. Elemento 4

La lista debe ir desde 1 hasta 4

1. Elemento 1 2. Elemento 2 3. Elemento 3 4. Elemento 4

La lista debe ir desde 33 hasta 36

33. Elemento 1 34. Elemento 2 35. Elemento 3 36. Elemento 4

La lista debe ir desde 100 hasta 97 (descendente)

100. Elemento 1 99. Elemento 2 98. Elemento 3 97. Elemento 4
